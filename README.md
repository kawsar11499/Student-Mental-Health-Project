# Student Mental Health

## Description
This project investigates the relationship between student stress, sleep, and academic pressure, with the goal of identifying actionable insights for university welfare departments.

## Dataset
- Source: [Insert dataset source link]
- Features: demographics, academic performance, stress levels, sleep patterns

## Tools & Technologies

## Key Findings
- Poor sleep quality strongly predicts high stress.
- Students with high stress are more likely to disengage from academic activities.
- Recommendations include early intervention, personalized support programs, and wellness resources.
