# Student Mental Health

## Description
This project investigates correlations and trends between student stress, course studied, and GPA, with the goal of identifying actionable insights for university welfare departments.

## Dataset
- Source: https://www.kaggle.com/datasets/shariful07/student-mental-health
- CC0: Public Domain, dataset made available through shariful07

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
